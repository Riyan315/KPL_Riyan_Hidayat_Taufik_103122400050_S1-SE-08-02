## Deskripsi
Pada penugasan ini, dibuat sebuah API sederhana menggunakan Express.js untuk menampilkan data menu makanan. API memiliki beberapa endpoint, yaitu untuk menampilkan daftar kategori menu dan detail menu berdasarkan kategori. Selain itu, dokumentasi API dibuat secara otomatis menggunakan Swagger (OpenAPI) sehingga memudahkan pengguna dalam memahami dan menguji endpoint yang tersedia.
